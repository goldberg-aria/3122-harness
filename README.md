# 3122-harness (archived)

This is an early Rust prototype of what became **Engram**.

It explored AMCP-native coding harness ideas — model-neutral runtime, structured memory triggers, multi-provider routing, and verification loops — and directly informed Engram's final design.

> 👉 **Active project**: [nunchi-ai/engram](https://github.com/nunchi-ai/engram)
> 👉 **Protocol**: [nunchi-ai/amcp](https://github.com/nunchi-ai/amcp)

Kept public as a historical reference. No longer maintained.
